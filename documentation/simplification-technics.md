## Clarify identifiers

> **Conjecture**
> 
> If `s` is a valid code snippet containing a given identifier `i`, the action to rename `i` may be a simplification. 

* Example

```ts
// Code snippet s
function f(a: number): number {
	return a * 2;
}

// Code snippet s'
function multiplyByTwo(a: number): number {
	return a * 2;
}
```
